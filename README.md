# Extractive Text Summarization

Automatic Extractive Text Summarization using TF-IDF Frequency Analysis.

> [Project Demo](https://createsummary.herokuapp.com)

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve
Start the server

```sh
$ npm run dev
```
